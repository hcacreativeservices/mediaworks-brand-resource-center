How to choose a new URL

When choosing a URL address for a new webpage, always start with the HCAhealthcare.com/ address. Then think of a keyword to represent your new page and add it to the end of the URL. Starting with HCAhealthcare.com makes it clear that the new page is part of our enterprise brand—making it both easy to find and to trust.

For example, HCAhealthcare.com/nurses is a straightforward URL. It establishes our enterprise site as the main website and uses an appropriate keyword to represent the subpage. It also shows off the site’s architecture and is easy for users to find and remember.
