Contrast and accessibility

The appropriate use of color with various font sizes, UI components and key graphical objects ensures that all of our communications are clear and easy for everyone to read. When using colors from the HCA Healthcare color palettes, there should be enough contrast between the background and type or graphics. Type and graphics that are placed on white, dark or colored backgrounds must exceed the minimum contrast ratio for AA accessibility. Text smaller than 18pt/24px* should have a ratio of at least 4.5:1; larger text, over 18pt/24px*, at least 3:1. While various combinations can be created using our color palettes, be sure to work within accessibility standards. The examples below demonstrate correct color use for on-screen (RGB) applications. All the color combinations on this page pass the minimum required AA accessibility rating. For more information on color contrast testing, see snook.ca colour contrast or the webaim.org contrast checker.

*Note: Pixel size is approximate and will depend on the font, browser and OS being used.

Note: HCA Healthcare Orange and white may only be used for type combinations when the font is 14pt bold or above, as it has a lower contrast ratio and will not pass accessibility requirements otherwise.
